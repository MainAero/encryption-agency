# About
This is a simple example how encrypt communication between to parties with java.

## Install
`mvn install`

## Run ApplicationOne
`sh startApplicationOne.sh`

## Run ApplicationTwo
After ApplicationOne started, run:
`sh startApplicationTwo.sh`
